# Fresh-Eyes Review Archive

> Dated artifacts from each fresh-eyes review session. Append-only.
> Procedure: [`../fresh-eyes-playbook.md`](../fresh-eyes-playbook.md).

Each file in this directory is the output of one invocation of the
fresh-eyes playbook — a 1,500-3,000 word review at a specific point in
time. Reviews are not edited after the fact; if a recommendation is
revisited or revised, that goes in a later review.

## How to read

Reviews are written cold against the project's then-current state.
Recommendations from a 6-month-old review may be obsolete; treat them as
historical context, not current guidance. The current state lives in
[`../../rd/pipeline.md`](../../rd/pipeline.md) and the candidate files.
